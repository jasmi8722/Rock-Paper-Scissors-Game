# Rock, Paper, Scissors Game

A fun and interactive Rock, Paper, Scissors game built with vanilla JavaScript, HTML, and CSS. Play against the computer and keep track of your score!

## Features

- **Interactive Gameplay**: Click on rock, paper, or scissors to make your move
- **Score Tracking**: Keep track of your wins and the computer's wins in real-time
- **Visual Feedback**: Get instant feedback on whether you won, lost, or drew the game
- **Responsive Design**: Clean and intuitive user interface with image-based choices
- **Computer AI**: The computer randomly selects its move for each round

## How to Play

1. Open `index.html` in your web browser
2. Click on one of the three choices: **Rock**, **Paper**, or **Scissors**
3. The computer will automatically make its choice
4. View the result message and updated score
5. Continue playing as many rounds as you like!

## Game Rules

- **Rock** beats **Scissors**
- **Scissors** beats **Paper**
- **Paper** beats **Rock**
- If both choose the same option, it's a **Draw**
